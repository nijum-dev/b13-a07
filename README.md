# KeenKeeper

## 📌 Project Description

Friends Keeper is a simple and clean web application that helps you track and maintain meaningful relationships. It allows you to manage your friends, monitor when you last contacted them, and stay consistent with your social connections.

---

## 🚀 Technologies Used

* **Next.js** (App Router)
* **React.js**
* **Tailwind CSS**
* **JSON (Static Data Handling)**

---

## ✨ Key Features

### 1. 👥 Friend Management

View a list of friends with profile pictures, tags, and personal details in a clean card layout.

### 2. ⏳ Contact Tracking

Track how many days have passed since you last contacted each friend, helping you stay connected.

### 3. 🎯 Status Indicators

Each friend is categorized as:

* **On Track** ✅
* **Almost Due** ⚠️
* **Overdue** ❗
  This makes it easy to prioritize who needs attention.
